How to Deploy on GitHub Pages

1. Create a new GitHub repo, such as audrey-vocab-practice.

2. Save this file as index.html.

3. Upload it to the repo root and push.

4. In GitHub: Settings → Pages → Deploy from a branch.

5. Choose the main branch and root folder. GitHub will publish the app after a short wait.
